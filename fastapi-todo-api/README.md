\# Simple FastAPI TODO API



Простой REST API для управления задачами (TODO list).  

Написан на \*\*FastAPI\*\*, данные хранятся в памяти. Отлично подходит для изучения основ FastAPI, тестирования через Swagger или Postman.



\## Возможности



\- Получить все задачи

\- Получить одну задачу по ID

\- Создать новую задачу

\- Полностью обновить задачу

\- Удалить задачу

\- Автоматическая документация Swagger UI и ReDoc
