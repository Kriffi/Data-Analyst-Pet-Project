
# Data-Analyst-Pet-Project

Добро пожаловать в репозиторий моих учебных и пет-проектов по анализу данных.  
Здесь собраны работы, демонстрирующие мой прогресс и навыки в области очистки данных, ETL, визуализации и бизнес-аналитики.

---

## Проекты

| Название проекта | Описание | Стек |
| :--- | :--- | :--- |
| [Исследование надёжности клиентов](https://github.com/Kriffi/Data-Analyst-Pet-Project/tree/main/Исследование%20надёжности%20клиентов) | Выявление факторов, влияющих на своевременное погашение кредитов. Проведена полная предобработка данных (пропуски, дубликаты, аномалии), исследованы зависимости между характеристиками заёмщиков и наличием просрочки. Результат — портрет надёжного и рискованного клиента для кредитного скоринга. | `Python` `Pandas` `NumPy` `Jupyter Notebook` |
| [Анализ Music Platform](https://github.com/Kriffi/Data-Analyst-Pet-Project/tree/main/Music%20Platform) | Полный цикл сбора данных через iTunes API, очистка и структурирование, загрузка в PostgreSQL, аналитические запросы. На примере дискографии Луи Армстронга получены инсайты о популярности, альбомах и треках. | `Python` `Requests` `SQL (PostgreSQL)` `Jupyter Notebook` `Git` |
| [Исследование данных стран](https://github.com/Kriffi/Data-Analyst-Pet-Project/tree/main/Fertility%20analysis) | ETL-консолидация социально-экономических показателей 224 стран в Power BI. Исправлены системные ошибки (десятичный разделитель, опечатка региона). Применён нечёткий LEFT JOIN (сохранность 98% записей). Построен интерактивный дашборд: карты, топ-10, регрессионный анализ (R²=0,76 между медианным возрастом и фертильностью). | `Power BI` `Power Query` `Python (pandas)` `SQL` `Fuzzy Join` |
| [Анализ car classification model](https://github.com/Kriffi/Data-Analyst-Pet-Project/tree/main/car%20classification%20model) | Этот проект посвящен подготовке и анализу набора данных о подержанных автомобилях, выставленных на продажу в США. Конечная цель — создать модель классификации, которая будет определять ценовую категорию (низкая, средняя, высокая) подержанного автомобиля на основе его характеристик. В этом отчете рассказывается о первичном изучении данных, определении переменных, работе с пропущенными значениями и анализе распределения, которые служат основой для этапа моделирования. |   `Python (pandas)` `SQL` ` matplotlib` |

## 🛠️ Общий стек

- **Языки:** Python (pandas, numpy, requests), SQL  
- **Инструменты:** Jupyter Notebook, PostgreSQL, Power BI, Git  
- **Навыки:** ETL, очистка данных, API, реляционные базы данных, визуализация, построение дашбордов

## 📌 Как использовать

Каждый проект находится в своей папке и содержит:
- Jupyter Notebook с кодом и пояснениями
- Файлы данных (если разрешено лицензией)
- README с описанием цели, хода работы и выводов
- Дашборд (для Power BI — .pbix файл)
